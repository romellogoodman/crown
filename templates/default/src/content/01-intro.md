# Introduction

Welcome to your first Crown book! Crown is a modern framework for creating print-quality PDFs using PrinceXML.

## Features

- **Markdown-based content** - Write your content in simple Markdown
- **Hot reload** - See changes instantly in your browser
- **Template system** - Use Handlebars templates for consistent layouts
- **Professional output** - Generate print-ready PDFs with PrinceXML

## Getting Started

Edit the markdown files in `src/content/` to add your own content. The files are automatically sorted by the `order` property in the frontmatter.

Try changing some text and watch the browser automatically refresh with your changes!
